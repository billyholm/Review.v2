# ReviewPeak

ReviewPeak hjälper småföretagare att få fler Google-recensioner via SMS.

## Teknologier

- **Next.js 14** - React framework med App Router
- **TypeScript** - Type-safe JavaScript
- **Tailwind CSS** - Utility-first CSS framework
- **Lucide React** - Ikonbibliotek

## Kom igång

### Installation

```bash
npm install
```

### Utvecklingsserver

```bash
npm run dev
```

Öppna [http://localhost:3000](http://localhost:3000) i din webbläsare.

### Bygga för produktion

```bash
npm run build
npm start
```

## Projektstruktur

- `/app` - Next.js App Router sidor och layouts
- `/components` - Återanvändbara React-komponenter
- `/public` - Statiska filer

## Sidor

- `/` - Landing page
- `/onboarding` - Onboarding-formulär för företagsinformation
- `/send-sms` - SMS-sändningsformulär
